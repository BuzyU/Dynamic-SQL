# Dynamic-SQL

A modern web application for managing hierarchical categories, systems, and records with Firebase integration.

## Features 

- 🔐 Secure Authentication with Firebase
- 📁 Hierarchical Category Management
- 🔄 Real-time Updates
- 🌗 Dark/Light Theme Support
- 📱 Responsive Design
- 📊 Systems Overview
- 🗂️ Record Management
- 🏷️ Custom Fields Support
- 🔗 Relationship Management
- 📚 Documentation with ER Diagrams

## Tech Stack

- Firebase (Authentication & Firestore)
- Tailwind CSS for styling
- Modern JavaScript (ES6+)
- React (for Documentation)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/BuzyU/Dynamic-SQL.git
```

2. Navigate to the project directory:
```bash
cd Dynamic-SQL
```

3. Set up Firebase:
   - Create a new Firebase project
   - Copy your Firebase config to `config.js`
   - Enable Authentication and Firestore

4. Open `index.html` in your browser

## Structure 

- `index.html` - Landing page and authentication
- `systems.html` - Systems management
- `dashboard.html` - Category management
- `documentation.html` - ER diagrams and documentation
- `records.html` - Record management interface
